# User Guide

## Getting Started

Open `index.html` or the deployed GitHub Pages URL. New users are sent to Simulation Selection. Returning users are sent to the dashboard.

## Completing a Simulation

1. Select a simulation.
2. Read the intro screen, then click **Start simulation** (or **Resume simulation** if you're returning to one in progress) when you're ready - the timer does not start until you click this button.
3. Inspect the example message or page shown for the first question. Click visible suspicious details - or press Tab to focus one and Enter/Space to select it - to record warning signs.
4. Select an answer, then read the feedback and choose **Next question**. A new, different example loads automatically for each of the three questions, so read each one freshly rather than assuming it repeats the last one.
5. Review the running score after each answer.
6. Finish the simulation, continue to another exercise, or open the updated report.

Each answer is saved immediately in LocalStorage. Refreshing or navigating away does not remove previously submitted answers.

Question order is randomized for each new attempt. The total timer measures active simulation time and pauses while the page is closed. Correct answers earn up to 20 additional speed points. The selection page shows **Resume** for unfinished sessions; reopening the exercise restores the exact question, feedback phase, elapsed time, and running score. After completion, choose **New randomized attempt** to clear that session and reshuffle the questions.

An exercise can be repeated. The latest result replaces the prior result for that scenario.

## Final Exam

Once you've practiced the 12 individual categories, **Final Exam** on the Simulation Selection page (shown last, with a gold capstone badge) pulls one question from every category into a single 12-question assessment. It works exactly like a regular simulation - timer gated behind Start, one example at a time, immediate feedback - just spanning every category instead of one. Completing the Final Exam is required for the certificate, in addition to completing all 12 individual categories.

## Training Center

Read each lesson and choose **Mark as complete**. Completion contributes to overall progress and lowers the calculated risk score.

## Dashboard and Report

The dashboard summarizes points, completion, risk, performance, and recent activity. The report page provides per-scenario results and recommendations. Choose **Export PDF** to download a dated report generated entirely in the browser.

## Appearance and Mobile Navigation

Use the moon or sun button in the top bar to switch themes. The choice persists. On smaller screens, use the menu button to open the sidebar and tap outside it to close it.

## Resetting Progress

Open Score & Report and choose **Reset**. Confirming removes all simulation, lesson, and activity records. The theme preference remains unchanged.
